


课程介绍
本视频涵盖MySQL核心技术主要知识点，每节知识配套对应练习。主要包含数据库和表的常用操作、约束、视图、存储过程和函数、流程控制结构以及综合运用各种命令实现数据的增删改查操作。本课程的目标是“看得懂、学得会、做得出”，为后续的学习打下夯实的基础。
